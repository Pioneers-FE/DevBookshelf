## 36장 디스트럭처링 할당

구조화된 배열과 같은 이터러블 또는 객체를 비구조화하여 1개 이상의 변수에 개별적으로 할당하는 것

### 36.1 배열 디스트럭처링 할당

배열 디스트럭처링 할당의 대상은 이터러블이여야 한다. 할당 기준은 배열의 인덱스

```jsx
const [a, b] = [1, 2];
// 1 2

const [c, d] = [1];
// 1 undefined

const [e, f] = [1, 2, 3];
// 1 2
```

### 36.2 객체 디스트럭처링 할당

```jsx
var user = { first: "tmp", last: "tmp2" };

var firstName = user.first;
var lastName = user.last;
```

할당 기준은 프로퍼티 키
